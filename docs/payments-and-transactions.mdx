에이전트가 지갑을 가지고 나면, 두 가지 핵심 기능이 열립니다: 서비스에 대한 **결제**와 온체인 **액션** 실행입니다.

## x402: 요청별 결제 프로토콜

x402는 Coinbase가 개발한 오픈 결제 프로토콜로, HTTP를 통해 자동으로 스테이블코인 결제가 이루어집니다. API 키나 구독 없이, 에이전트가 사용한 만큼만 즉시 지불할 수 있습니다.

### 작동 방식

1. 에이전트가 유료 API 엔드포인트에 요청을 보냅니다
2. 서버가 HTTP `402 Payment Required` 상태 코드와 함께 결제 세부사항을 반환합니다
3. 에이전트가 자동으로 USDC 결제를 생성합니다
4. 결제 증명과 함께 요청을 재전송합니다
5. 서버가 결제를 검증하고 데이터를 반환합니다

```typescript
// x402 결제 흐름 예시
const response = await fetch("https://api.example.com/premium-data");

if (response.status === 402) {
  const paymentDetails = await response.json();
  
  // 에이전트가 자동으로 결제 생성
  const payment = await wallet.createPayment({
    amount: paymentDetails.amount,
    recipient: paymentDetails.recipient,
    currency: "USDC",
  });
  
  // 결제와 함께 재요청
  const paidResponse = await fetch("https://api.example.com/premium-data", {
    headers: { "X-PAYMENT": payment.payload },
  });
}
```

### x402의 장점

- **자율 결제**: 사람의 개입 없이 에이전트가 직접 결제합니다
- **사용량 기반 과금**: 실제 사용한 만큼만 지불합니다
- **최소 설정**: 미들웨어 한 줄로 통합 가능합니다
- **즉시 정산**: 스테이블코인 기반으로 즉각적인 결제가 이루어집니다

## Skill: 사전 구축된 온체인 액션

Skill은 에이전트가 복잡한 트랜잭션 로직을 직접 구축하지 않고도 수행할 수 있는 사전 구축된 온체인 작업입니다.

### 주요 Skill

| Skill | 설명 |
| --- | --- |
| **Authenticate** | 지갑 연결 및 인증 |
| **Fund** | 에이전트 지갑에 자금 입금 |
| **Send** | 토큰 및 스테이블코인 전송 |
| **Trade** | 토큰 스왑 실행 |
| **Earn** | DeFi 프로토콜에서 수익 창출 |

### Skill 통합 방법

AgentKit의 `agent-wallet-skills` 리포지토리를 통해 skill을 에이전트에 통합할 수 있습니다:

```bash
# skill 리포지토리 클론
git clone https://github.com/coinbase/agent-wallet-skills
```

### 커스텀 Skill 생성

기본 제공되는 skill 외에도, 에이전트의 특정 요구사항에 맞는 커스텀 skill을 만들 수 있습니다:

```typescript
// 커스텀 DeFi skill 예시
const customSkill = {
  name: "YIELD_FARM",
  description: "Base 프로토콜에서 자동으로 yield farming을 수행합니다",
  handler: async (params) => {
    // yield farming 로직 구현
  },
};
```

## 가스리스 트랜잭션

Base에서는 에이전트가 가스비 없이 트랜잭션을 실행할 수 있습니다. 이를 통해 에이전트가 가스 부족으로 중단되지 않고 24/7 자율적으로 운영될 수 있습니다.

<Tip>
테스트 시에는 Base Sepolia 테스트넷을 사용하세요. 실제 자금 없이 모든 결제 및 트랜잭션 기능을 테스트할 수 있습니다.
</Tip>

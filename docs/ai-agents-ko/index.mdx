Base는 여러분의 AI 에이전트가 독립적인 경제 주체로 활동하는 데 필요한 도구를 제공합니다. 자금을 보유하고 사용할 수 있는 지갑, 다른 에이전트와 서비스가 신뢰할 수 있도록 하는 신원 표준, 요청별 결제(pay-per-request) API를 위한 결제 프로토콜, 그리고 에이전트들이 서로를 찾을 수 있는 디스커버리 레이어를 제공합니다.

## 구축할 수 있는 것들

- **트레이딩 에이전트** — 시장 상황을 모니터링하고 skill과 자금이 입금된 지갑을 사용하여 자동으로 토큰 스왑을 실행합니다
- **결제 에이전트** — x402 요청별 결제 프로토콜을 사용하여 애플리케이션을 대신하여 프리미엄 API 데이터 비용을 지불합니다
- **디스커버리 가능한 서비스** — 다른 에이전트가 찾고, 검증하고, 비용을 지불하여 사용할 수 있습니다. 사람의 통합 작업이 필요 없습니다
- **멀티채널 어시스턴트** — Discord, Telegram 또는 웹에서 지갑을 관리하고, 질문에 답하며, 온체인 액션을 실행합니다

## 구성 요소들이 어떻게 연결되는가

1. **프레임워크 선택** — 에이전트를 구축하고 실행하는 방법을 선택합니다 (자체 호스팅 SDK, 오픈소스 어시스턴트, 또는 관리형 API)
2. **지갑 추가** — 에이전트에게 스테이블코인 보유, 결제 전송, 트랜잭션 서명 기능을 부여합니다
3. **결제와 skill 사용** — 에이전트가 x402를 통해 서비스에 대한 비용을 지불하고, 사전 구축된 skill을 통해 온체인 액션을 수행합니다
4. **신원 등록** — 다른 에이전트와 서비스가 발견하고 검증할 수 있도록 공개 디렉토리에 에이전트를 등록합니다
5. **Agent App 구축 또는 연결** — 에이전트를 주요 사용자로 설계된 서비스를 만들거나, 기존 서비스에 에이전트를 연결합니다

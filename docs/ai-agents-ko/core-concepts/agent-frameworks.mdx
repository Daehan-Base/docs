## 옵션 비교

| | BANKR | OpenClaw | Agent SDK |
|---|---|---|---|
| **개요** | 관리형 에이전트 API, OpenClaw용 스킬 및 지갑 플러그인 제공 | Discord, Telegram, 웹 등 다양한 채널에서 실행되는 오픈소스 개인 AI 어시스턴트 | 동작과 연동을 완전히 제어할 수 있는 커스텀 에이전트 개발용 툴킷 |
| **언어** | Agent API는 REST(언어 무관), OpenClaw 플러그인은 TypeScript | TypeScript | TypeScript / Python |
| **호스팅** | Agent API는 BANKR이 관리, OpenClaw 플러그인은 자체 호스팅 에이전트에서 실행 | 자체 호스팅 | 자체 호스팅 |
| **적합한 경우** | 인프라 없이 온체인 기능을 추가할 때는 Agent API, 자체 에이전트에 온체인 스킬을 추가할 때는 OpenClaw 플러그인 | 플러그인 지원이 있는 멀티채널 챗봇을 운영할 때 | 로직, 연동, 배포를 완전히 제어해야 하는 프로덕션 에이전트 |

## BANKR

BANKR은 Base에서 에이전트를 다루는 두 가지 방법을 제공합니다. **BANKR Agent API**는 REST API를 통해 상호작용하는 호스팅 에이전트입니다. 프롬프트를 보내면 결과를 받을 수 있으며, 에이전트를 직접 빌드하거나 배포하지 않고도 트레이드 실행이나 잔액 조회가 가능합니다. 기존 애플리케이션에 온체인 에이전트 기능을 추가하는 가장 빠른 방법입니다.

BANKR은 **OpenClaw용 스킬 및 지갑 플러그인**도 제공합니다. OpenClaw로 에이전트를 개발 중이라면 BANKR의 스킬 패키지를 설치하여 관리형 API 없이도 지갑 기능, 토큰 스왑, 기타 온체인 액션을 에이전트에 추가할 수 있습니다.

인프라 없이 온체인 에이전트 기능을 원한다면 BANKR Agent API를, 자체 호스팅 에이전트에 BANKR의 온체인 스킬을 추가하고 싶다면 BANKR OpenClaw 플러그인을 사용하세요.

[BANKR Agent API 시작하기 →](https://docs.bankr.bot/getting-started/quick-start)

[OpenClaw용 BANKR 스킬 둘러보기 →](https://github.com/BankrBot/openclaw-skills)

## OpenClaw

OpenClaw는 여러 메시징 플랫폼에 기본으로 연결되는 오픈소스 AI 어시스턴트입니다. 스킬(에이전트가 수행할 수 있는 사전 빌드된 액션)을 추가하는 플러그인 시스템을 갖추고 있으며, 채널 간 메시지 수신·처리·응답을 위한 인프라를 자체적으로 처리합니다.

Discord, Telegram, 웹 등 여러 채널에서 대화형 에이전트를 운영하고 커뮤니티 플러그인으로 확장하고 싶다면 OpenClaw를 사용하세요.

[OpenClaw 시작하기 →](https://docs.openclaw.ai/start/getting-started)

## Agent SDK

Coinbase의 Agent SDK는 <Tooltip tip="탈중앙화 거래소나 대출 플랫폼과 같은 블록체인 기반 프로토콜 및 애플리케이션">온체인 서비스</Tooltip>와 상호작용할 수 있는 에이전트를 만들기 위한 빌딩 블록을 제공합니다. 에이전트 로직을 직접 작성하고, 도구를 정의하며, 자체 인프라에 배포합니다. 프롬프트 처리 방식부터 호출할 서비스까지 에이전트의 모든 동작을 제어할 수 있는 가장 유연한 옵션입니다.

프로덕션 에이전트를 구축하고 아키텍처를 완전히 제어하고 싶다면 Agent SDK를 사용하세요.

[Agent SDK 시작하기 →](https://docs.cdp.coinbase.com/agent-kit/welcome)

<Tip>
**선택 가이드:** 에이전트를 직접 만들지 않고 API로 에이전트를 호출하려면 **BANKR Agent API**를, OpenClaw 에이전트에 온체인 기능을 추가하려면 **BANKR 스킬 플러그인**을 사용하세요. 플러그인이 있는 멀티채널 챗봇이 필요하다면 **OpenClaw**를, 로직·연동·배포를 완전히 제어하고 싶다면 **Agent SDK**를 사용하세요.
</Tip>

## 왜 한국어 번역을 만들었나요?

한국은 전 세계에서 가장 활발한 Web3 및 블록체인 개발자 커뮤니티 중 하나를 보유하고 있습니다. 한국의 개발자들은 이미 DeFi, NFT, 온체인 게임 등 다양한 영역에서 혁신을 이끌어 왔습니다. 하지만 영어로만 제공되는 기술 문서는 많은 한국 개발자들에게 진입 장벽이 되어 왔습니다.

저희는 Base 위에서 에이전트를 개발하려는 한국 개발자들이 언어의 장벽 없이 바로 시작할 수 있도록, Base 에이전트 문서를 한국어로 번역하기로 결정했습니다.

---

## 이 번역이 중요한 이유

### 더 많은 개발자에게 문을 열다

복잡한 에이전트 아키텍처, 스마트 컨트랙트 통합, 온체인 자동화 개념은 모국어로 읽을 때 훨씬 빠르게 이해됩니다. 기술 개념을 영어로 먼저 해석하고 다시 이해하는 과정 없이, 핵심 아이디어에 바로 집중할 수 있습니다.

### 한국 Web3 생태계를 강화하다

한국의 개발자, 스타트업, 해커톤 팀들이 Base 위에서 에이전트를 더 빠르게 프로토타입하고 제품을 출시할 수 있도록 지원합니다. 언어 접근성은 생태계 성장의 핵심 요소입니다.

### 커뮤니티 중심의 성장

이 번역은 커뮤니티의 기여로 이루어졌습니다. Base를 사랑하는 한국 개발자들이 직접 참여하여 더 정확하고 자연스러운 표현을 만들어 나가고 있습니다. 번역 개선에 기여하고 싶다면 언제든지 [Pull Request를 보내 주세요](https://github.com/Daehan-Base).

---

## 에이전트란 무엇인가요?

Base 위의 에이전트는 온체인에서 자율적으로 행동하는 프로그램입니다. 에이전트는 다음과 같은 작업을 수행할 수 있습니다.

- 지갑을 통해 트랜잭션을 직접 실행
- 온체인 데이터를 읽고 조건에 따라 반응
- 다른 스마트 컨트랙트와 상호작용
- 사용자 대신 반복적인 온체인 작업 자동화
